Using lftp with TLS Enabled

// under ftp sysadm

Installation

FreeBSD

portmaster ftp/lftp Enable the TLS by checking the box in the configure dialog

Arch Linux

abs mkdir ~/abs cp -r /var/abs/lftp ~/abs cd ~/abs/lftp

We need to modify the PKGBUILD of lftp

...
depends=('gcc-libs' 'readline' 'openssl' 'expat' 'sh')
...
build() {
    cd ${pkgname}-${pkgver}
    ./configure --prefix=/usr \
        --without-gnutls \
        --with-openssl \
        --without-included-regex \
        --disable-static
    make
}
...

makepkg -s pacman -U lftp-4.5.5-1-x86_64.pkg.tar.xz

Mac OS X

First, update Homebrew formula by brew update. When it is done, just install it through brew install lftp.
